## Arrow tooltips

You can use the `arrow` prop to give your tooltip an arrow indicating which element it refers to. Arrow direction depends on tooltip position.

<Canvas of={tooltipV2.ArrowTooltips} withToolbar layout='centered'/>

## Showing and hiding

The tooltip is normally shown immediately when the user's mouse hovers over the element, and hides immediately when the user's mouse leaves. A delay in showing or hiding the tooltip can be added through the `enterDelay` and `leaveDelay` props.

<Canvas of={tooltipV2.ShowingAndHiding} withToolbar layout='centered'/>

## Distance from anchor

You can use the `offsetX` and `offsetY` prop to control the tooltip distance from the anchor.

<Canvas of={tooltipV2.Offset} withToolbar layout='centered'/>
